Installation of New Dictionaries:

1 - Dictionaries can be downloaded for free from www.aspspellcheck.com/downloads/
2 - Unzip 
3 - Copy the .dic file to this folder /ASPSpellCheck/dictionaries/

Adding New Words:
1 - Add 1 word per line to /ASPSpellCheck/dictionaries/custom.txt

Removing Words:
1 - Add 1 word per line to /ASPSpellCheck/dictionaries/language-rules/banned-words.txt


Happy Coding
